# STATS 608 Enhanced Project Bundle

Contents:

- `STATS608_project_wrapup.ipynb` — executed notebook with explanations, tables, and figures.
- `STATS608_project_wrapup.html` — browser-friendly export of the notebook.
- `enhanced_pipeline.py` — enhanced experiment code used for the notebook.
- `primary_results.csv` — main experiment outputs.
- `stress_results.csv` — harder conditioning experiment outputs.

- `primary_summary.csv`, `stress_summary.csv`, `block_sweep_summary.csv` — compact summaries.

The notebook assumes the bundle directory is intact.
